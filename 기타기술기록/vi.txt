세 가지 모드가 있음
0. 선택 모드 (핵심)
1. 단축키 입력 모드 (default)
2. 텍스트 입력 모드	
	진입방법: i, a, shift + i, shift + a
		i: 커서 앞쪽에서 시작
		a: 커서 뒷쪽에서 시작
		shift + i : 줄 맨 앞에서 시작
		shift + a : 줄 맨 뒤에서 시작
	텍스트 입력 모드 종료: esc
	여기선, 이동하지 말 것. 즉, 방향키 써서 이동하지 말란 뜻이다.
단축키 입력 모드에서 방향키 눌러보기
	h	왼쪽
	j	아래
	k	위
	l	오른쪽

단어 단위 좌우 이동하기: 
	w 앞으로
	b 뒤로

문단 단위 이동하기
	shift + {
	shift + }


*** 이제 선택(비쥬얼 모드). 가장 중요하게 쓰는 키들
	v: 선택 모드
	shift+v: 줄 단위 선택 모드
	y: 복사
	p: 붙여넣기
	d: 선택부분 지우기
	x: 선택부분 오려두기

* 덜 중요하지만 알아두면 유용한 것.
	선택 후 shift + < 또는 > : 들여쓰기, 내어쓰기
	x: 글자 지우기
	r: 글자 수정. 누른 후 원하는 글자
	shift + r: 수정(insert) 키
	shift + h: 화면 맨 위
	shift + l: 화면 맨 아래
	shift + m: 화면 한 가운데
	ctrl + u : up. 반 페이지 위로 이동
	ctrl + d : down. 반 페이지 아래로 이동
